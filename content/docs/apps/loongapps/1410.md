{{< issue 运行时突然崩溃 >}}
目前腾讯对内置 libvips 的构建方法有问题：腾讯在 37xxx 版本开始在 libvips 中静态链接了 GLib，而静态链接的老版本 GLib 和新世界系统中的新版 GTK 交互时触发了 assert

解决办法：
```bash
sudo rm -v /opt/QQ/resources/app/sharp-lib/libvips-cpp.so.42
sudo apt install libvips42
```

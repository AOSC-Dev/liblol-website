经测试，需要系统安装 openssl-1.1 才能正常运行。
AOSC OS下打开终端，执行：
```bash
oma install openssl-1.1
```
即可解决
deepinV23下则为 libssl-1.1 ：
```bash
sudo apt install libssl-1.1
```

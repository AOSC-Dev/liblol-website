{{< issue "使用LG110显卡的用户可能会出现白屏并闪退" >}}

具体表现为：

从图标启动时，会先白屏，然后直接退出，没有任何提示。  

从终端启动时，报以下错误，然后直接退出：
```
clone: Invalid argument
[1]    2998 trace trap (core dumped)  ./qqmusic
```  
可尝试加上以下参数：
```
--in-process-gpu
```
从终端使用以下命令启动QQ音乐：
```
/opt/qqmusic/qqmusic --no-sandbox %U --in-process-gpu
```

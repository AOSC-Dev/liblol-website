{{< issue "使用LG110显卡的用户可能会出现白屏和闪退" >}}

从图标启动时会出现白屏。从终端使用默认命令启动时，会报以下错误：
```
Gtk-Message: 17:43:06.258: Failed to load module "appmenu-gtk-module"
Gtk-Message: 17:43:06.407: Failed to load module "colorreload-gtk-module"
Gtk-Message: 17:43:06.408: Failed to load module "window-decorations-gtk-module"
(electron) The default value of app.allowRendererProcessReuse is deprecated, it is currently "false".  It will change to be "true" in Electron 9.  For more information please check https://github.com/electron/electron/issues/18397
QQ login
cookie set success
cookie set success
[93472:0214/174314.103271:FATAL:gpu_data_manager_impl_private.cc(1034)] The display compositor is frequently crashing. Goodbye.
[1]    93472 trace trap (core dumped)  /opt/qqmusic/qqmusic --no-sandbox %U
```  
此时，可尝试加上以下参数：
```
--in-process-gpu
```
即从终端使用以下命令启动QQ音乐：
```
/opt/qqmusic/qqmusic --no-sandbox %U --in-process-gpu
```
此时QQ音乐可启动。

{{< /issue >}}

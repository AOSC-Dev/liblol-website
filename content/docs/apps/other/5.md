### 额外操作
安装后直接打开钉钉会闪退，需要额外操作。以下为AOSC OS下的操作：

1. 进入钉钉安装目录：
```
cd /opt/apps/com.alibabainc.dingtalk/files/<版本号>/
```
本页面更新时最新版本为8.1.0.6011301，直接使用以下命令：
```
cd /opt/apps/com.alibabainc.dingtalk/files/8.1.0-Release.6011301/
```

2. 启动钉钉：

```
./com.alibabainc.dingtalk
```
此时会报错，记下缺少的库文件。  
如果是8.1.0.6011301，可能会缺少liblber-2.4.so.2  

3. 下载和提取缺少的库文件：

如果是8.1.0.6011301版本的钉钉，在以下链接下载。
```
https://pkg.loongnix.cn/loongnix/20/pool/main/o/openldap/libldap-2.4-2_2.4.47%2Bdfsg-3%2Bdeb10u2.lnd.2_loongarch64.deb
```

然后，使用`dpkg -x`命令解压下载的包，从中找到缺失的库文件`liblber-2.4.so.2 `。

将该文件拷贝到以下目录：
```
 /opt/lol/lib/loongarch64-aosc-linux-gnuow/ 
```
4. 修改启动脚本。
```
vim  /opt/apps/com.alibabainc.dingtalk/files/Elevator.sh
```
内容替换为
```
#!/bin/sh
cd /opt/apps/com.alibabainc.dingtalk/files/<版本号>/
./com.alibabainc.dingtalk
```
例如如果是8.1.0.6011301版本，直接替换为：
```
/opt/apps/com.alibabainc.dingtalk/files/8.1.0-Release.6011301/
```
随后，正常打开钉钉即可运行。
